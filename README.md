# こまちゃん画像変換

## ソフトウェアの概要

「こまちゃん画像変換」は、複数の画像ファイルをまとめて変換できる Windows 用の画像変換ツールです。

画像形式の変換、リサイズ、トリミング、回転・反転、文字列や別画像の合成、連番付きファイル名の生成などを、1つのメイン画面から設定できます。

設定はカレントディレクトリの `settings.config` に保存されます。  
通常の変換設定は「設定を保存」ボタンを押したときに保存され、次回起動時に読み込まれます。  
ウィンドウの位置とサイズは終了時に自動保存され、次回起動時に復元されます。

インストールされていない場合、インストールを促す画面が表示されます。

## ソフトウェアの基本的な使い方

1. 変換したい画像ファイルを、右側の「変換対象ファイル」リストへドラッグアンドドロップします。
2. 「保存先ディレクトリ」を指定します。
3. 必要に応じて、保存形式、画像名、リサイズ、合成、回転・反転などを設定します。
4. 「プレビュー」ボタンで変換結果を確認します。
5. 右側リスト下の「変換開始」ボタンを押すと変換が開始されます。
6. 変換中はステータスバーのプログレスバーに進捗が表示されます。
7. 変換が完了すると、ステータスバーに完了件数が表示されます。

## メイン画面説明

### 共通操作

#### 設定を保存

現在の設定内容を `settings.config` に保存します。

自動保存されるのはウィンドウ位置とサイズのみです。変換設定を次回起動時にも使いたい場合は、このボタンで保存してください。

#### プレビュー

選択中の画像に現在の設定を適用した結果を、別ウィンドウで表示します。

プレビュー画像がウィンドウより大きい場合はスクロールできます。スクロール可能な場合は手のひらカーソルになり、画像をドラッグして表示位置を移動できます。

#### 画像番号（連番）

生成ファイル名に `%cn%` を使用した場合に付加される連番です。

テキストボックスの数値を変更して「変更」ボタンを押すと、以降の変換で使用される連番が変わります。同名ファイルがある場合は上書きされる可能性があります。

### 変換対象ファイル

右側のリストに表示されている画像が変換対象です。

画像ファイルまたは画像を含むフォルダをドラッグアンドドロップすると、対応画像がリストへ追加されます。対応拡張子は `.jpg`, `.jpeg`, `.png`, `.gif`, `.bmp`, `.tif`, `.tiff` です。

#### 変換開始

リスト内の画像を順番に変換します。

変換中はボタンが無効になり、二重実行を防ぎます。変換処理はUIとは別スレッドで実行されるため、大量の画像を変換している間も進捗表示が更新されます。

#### リスト保存

現在の変換対象リストを、カレントディレクトリの `list.csv` に保存します。

#### リスト読込

`list.csv` からファイルパスを読み込み、変換対象リストに追加します。

#### クリア

リストの内容をすべて削除します。実際の画像ファイルは削除されません。

#### 右クリックメニュー

リストを右クリックするとメニューが表示されます。

- リストから削除: 選択中の項目をリストから削除します。Deleteキーでも削除できます。
- リストからすべて削除: リスト内の項目をすべて削除します。

どちらもリストから外すだけで、実ファイルは削除しません。

## 「基本・合成」タブ

### 保存先ディレクトリ

変換後の画像を保存するフォルダを指定します。

保存先が空の場合は変換できません。初期値はカレントディレクトリ内の `ConvertedImages` です。

- 選択: フォルダ選択ダイアログを開きます。
- 開く: 保存先フォルダをエクスプローラーで開きます。フォルダが存在しない場合は作成してから開きます。

フォルダやファイルを保存先ディレクトリ欄へドラッグアンドドロップすると、そのフォルダパスを設定できます。ファイルをドロップした場合は、そのファイルがあるフォルダを設定します。

### 画像の名前

生成される画像ファイル名を指定します。

ファイル名に使用できない文字 `\ / : * ? " < > |` は `_` に置換されます。

使用できる特殊文字列は以下です。

- `%of%`: 元ファイル名
- `%t%`: 変換時点の日付と時刻 `yyyyMMddHHmmss`
- `%t1%`: 変換時点の日付 `yyyy_MM_dd`
- `%t2%`: 変換時点の時刻 `HH_mm_ss`
- `%t3%`: 変換時点の日付と時刻 `yyyy_MM_dd_HH_mm`
- `%cn%`: 連番

例: `%of%_%t%_R_%cn%`

「元のファイル名で保存」にチェックを入れると、画像の名前欄は無効になり、元画像と同じファイル名で保存します。

### 保存形式

保存形式を選択します。

- JPEG
- PNG
- GIF

JPEGを選択した場合は、JPEG品質を `0～100` の範囲で指定できます。数値が大きいほど高画質になり、初期値は `95` です。

### 画像のサイズ指定

画像サイズの変更方法を指定します。リサイズ時の補完方法は `HighQualityBicubic` 固定です。

#### 幅と高さを固定

指定した幅と高さに変換します。元画像の縦横比は維持されません。

#### 画像の高さを指定

指定した高さに合わせて変換します。幅は元画像の縦横比を維持して自動計算されます。

#### 画像の幅を指定

指定した幅に合わせて変換します。高さは元画像の縦横比を維持して自動計算されます。

#### 画像の比率を指定

元画像に対する倍率をパーセントで指定します。

#### 指定範囲内におさめる

指定した幅と高さの範囲内に収まるように縮小します。元画像が指定範囲内に収まっている場合は、元サイズのままです。

#### サイズは変更しない

元画像と同じサイズで出力します。

#### トリミング

左上からの座標 `トリミングX`, `トリミングY` と、幅・高さを指定して画像を切り抜きます。

#### 画像が指定サイズより大きいときにのみ縮小

幅指定または高さ指定のときに使用します。チェックが入っている場合、元画像が指定サイズ以下なら拡大せず元サイズのままにします。

### 合成

画像に文字列や別画像を重ねる設定です。

#### 透明度

文字列や別画像を合成するときの透明度を指定します。

右へ動かすほど濃く、左へ動かすほど薄くなります。文字色そのものは選択した色で保存されますが、透明度が100%未満の場合は背景画像と混ざって見えます。

#### 位置 X / 位置 Y

合成開始位置を指定します。

正の値は画像左上を基準にします。負の値は画像右下を基準にします。

#### 画像に文字列を合成

チェックすると、指定した文字列を画像に合成します。

#### 画像に合成する文字列

任意の文字列を指定します。

使用できる特殊文字列は以下です。

- `%of%`: 元ファイル名（拡張子なし）
- `%t9%`: 写真の撮影日時（JPEGのEXIF情報がある場合のみ）
- `%t1%`: 変換時点の日付 `yyyy/MM/dd`
- `%t2%`: 変換時点の時刻 `HH:mm:ss`
- `%t3%`: 変換時点の日付と時刻 `yyyy/MM/dd HH:mm`

#### 文字色を選択

色の設定ダイアログを開き、合成文字列の色を指定します。

#### フォントを選択

フォント選択ダイアログを開き、合成文字列のフォント、サイズ、スタイルを指定します。

#### 画像に別の画像を合成

チェックすると、指定した画像を元画像に合成します。

「選択」ボタンで合成画像をファイル選択ダイアログから指定できます。

#### 合成画像の大きさ（%）

合成する別画像の大きさを指定します。

`100%` は元サイズのまま、`50%` は半分の大きさ、`200%` は2倍の大きさで合成します。

#### 合成画像の位置プリセット

合成画像の配置をラジオボタンで指定できます。

- 手動: 位置X / 位置Y の値を使用します。
- 左上: X=0, Y=0
- 右上: 合成画像のサイズを考慮して右上に配置します。
- 左下: 合成画像のサイズを考慮して左下に配置します。
- 右下: 合成画像のサイズを考慮して右下に配置します。

右上、左下、右下を選択した場合、合成画像の拡大縮小後サイズを考慮して位置が決まります。

## 「詳細」タブ

### 設定変更時にプレビューを実施する

チェックすると、設定を変更したときにプレビューを自動更新します。

### ファイルが存在する場合、上書き確認

変換先に同名ファイルがある場合、上書きしてよいか確認します。

チェックがOFFの場合は、確認せず上書きします。

### ドラッグアンドドロップで即座に変換開始

チェックすると、画像をリストへドラッグアンドドロップした直後に変換を開始します。

### 画像回転・反転

変換時に画像を回転または反転します。

- なし
- 90度回転
- 180度回転
- 270度回転
- 左右反転
- 上下反転

### Exif / メタ情報設定

#### Exif情報を引き継ぐ（JPEGのみ）

JPEG画像のEXIF情報を、変換後のJPEGへ引き継ぎます。

#### PNGメタ情報を引き継ぐ（PNGのみ）

PNG画像のメタ情報を、変換後のPNGへ引き継ぎます。

WICで扱えない特殊なメタ情報がある場合は、画像保存を優先して通常のPNGとして保存します。
